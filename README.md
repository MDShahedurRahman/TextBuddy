# TextBuddy

TextBuddy is a small Python command-line text utility.  
It can analyze text, show top words, find word occurrences, replace words, and export a JSON report.

## Requirements
- Python 3.9+

## Setup
```bash
python -m venv .venv
source .venv/bin/activate   # macOS/Linux
# .\.venv\Scripts\Activate.ps1  # Windows PowerShell

python -m pip install --upgrade pip
pip install -r requirements.txt
